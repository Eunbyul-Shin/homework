# 첫 번째 회고

한 주동안 가장 어려웠던 것은 정해진 시간에 일어나 준비하는 거 였던 것 같습니다. 분명 오프라인 수업이 아닌 온라인 수업임에도 같은 한 자리에 꾸준히 앉아 있는 게 가장 어려웠던 것 같습니다. 오래 앉아 있기 위해서는 오히려 등을 펴고 조금 더 집중할 수 있는 공간을 만들어야 한다는 점을 깨달았습니다.

두 번째로 어려웠던 점은 git 관리였던 것 같습니다. 학교 내에서 프로젝트보다 자격증을 따고 전공 수업에 따라가는 것에 지쳐서 자주 경험해보지 못했었고, git 프로그램을 이용하여 한 번에 처리했고 merge 또한 총괄해서 수정해주는 친구가 있었기에 직접 해볼 생각을 하지 못했었습니다. 그저 기능을 잘 만드는 개발자가 되자, 라는 마음이 강했던 것 같습니다. 아예 기초부터 git 관리와 merge에 대해서 차근차근 순서를 배우니 왜 지금까지 시도해볼 생각은 못했던 것인지. 무지에 대한 두려움이 컸다고 생각합니다.

아직 1주차이지만, 회고에 대해서 서투른 점이 많습니다. 조금 더 함께 하는 인원들과 함께 회고하고 스스로 배워나가는 방법을 배우고 싶습니다.
